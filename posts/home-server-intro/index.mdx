## 홈서버를 사게된 계기

회사에서 동료 개발자와 이런저런 이야기를 나누던 중, 홈서버 이야기가 나왔다. "홈서버를 한번 사서 구축해보고 싶은데…" 하고 넌지시 말을 꺼내자, 동료는 [2cpu](https://www.2cpu.co.kr/)라는 중고 장비 거래 사이트를 알려주며 장비와 신뢰할 만한 판매자까지 추천해줬다.

그렇게 구매하게 된 장비가 Dell OptiPlex 3000 미니였다.

<Image src={dellOptiplex} alt="dell optiplex 장비" />

### 왜 Dell OptiPlex 3000 미니인가?

Dell OptiPlex 3000 미니는 홈서버 입문용으로 딱 좋은 사양을 갖추고 있다:

- 크기가 작아 공간 차지가 적다.
- 전력 소모가 적다.
- 합리적인 가격
- NAS와 웹서버를 운영하기에 괜찮은 성능

개인적으로 이 프로젝트를 시작한 목적은 **네트워크 인프라와 서버에 대한 이해도를 실전을 통해 높이는 것이다.** 이론으로만 알던 것들을 직접 손으로 만져가며 배우는 것만큼 확실한 학습은 없으니까.

## 정보 수집 전략 : AI 심층 리서치 활용

홈서버 구축은 생각보다 고려할 사항이 많다. 운영체제 선택부터 Docker 컨테이너 구성, 네트워크 설정까지... 혼자 모든 정보를 찾아보려면 시간이 오래 걸린다. 

그래서 ChatGPT 심층 리서치 기능을 적극 활용했다. 만약 만약 심층 리서치 한도를 다 사용했다면 Gemini Deep Research나 Grok DeepSearch 같은 대안도 좋다.

<Image src={deepResearch} alt="deep research" />

실제로 질문한 내용 : 

- ChatGPT에는 아래와 같이 간단하게 질문했다.

```jsx
dell optiplex 3000 미니 컴퓨터에 NAS 혹은 웹 서버 용으로 홈서버를 구축하려 한다. 
- Ubuntu 24.04 LTS Server 설치 
- Docker 활용
- 내 메인 컴퓨터에서 터미널 원격 접속
```

ChatGPT는 바로 답변을 주는 대신, 더 정확한 가이드를 제공하기 위해 추가 질문을 진행한다. 

```jsx
// 추가 질문
Dell OptiPlex 3000 미니 컴퓨터를 홈 서버로 구축하려는 계획을 잘 이해했어요. 다음 내용을 조금만 더 구체적으로 알려주시면, 구성과 도구 추천을 도와드릴 수 있어요:

1. NAS: 단순 파일 공유만 필요한가요, 아니면 미디어 서버(Plex, Jellyfin 등)도 고려하고 계신가요?
2. 웹 서버: 주 용도는 개발 테스트용인가요, 아니면 외부에 공개할 서비스 운영도 계획 중인가요?
3. Docker: 이미 어떤 컨테이너를 쓸지 구상하신 게 있나요? 예: Nginx, PostgreSQL, Nextcloud, etc.
4. 네트워크: 포트포워딩, DDNS, HTTPS 인증서 사용 계획도 있으신가요?
5. 성능 최적화나 보안 구성도 포함해서 도와드릴까요?

이 정보를 주시면 필요한 설정, 툴, Docker 구성 예시 등을 정확하게 맞춤 제공해드릴게요.
```

이런 질문에 답변하니, ChatGPT는 내 상황에 딱 맞는 맞춤형 가이드를 제공해줬다. 일부 맞지 않는 부분이 있었지만, **대부분의 내용이 실제로 적용 가능한 실용적인 정보**였다.

이 과정을 통해 홈서버 구축의 전체 흐름을 파악할 수 있고, 부분적인 과정을 다시 질문하며 학습할 수 있다.
