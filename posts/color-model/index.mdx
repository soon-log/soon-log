# 색상 모델

## RGB
Red(빨강), Green(초록), Blue(파랑) 세 가지 빛의 값을 조합하여 색을 표현하는 방식이다.
각 값은 0부터 255까지의 숫자를 가지며 이는 모니터와 같은 디지털 기기가 색상을 출력하는 방식과 직접적으로 연결되어 있다.

<Image src={RGB} alt="RGB 색상" />

```css
.btn-rgb-1 {
  background-color: rgb(255, 0, 0); /* Red */
}

.btn-rgb-2 {
  background-color: rgb(0, 255, 0); /* Green */
}

.btn-rgb-3 {
  background-color: rgb(0, 0, 255); /* Blue */
}

.btn-rgb-4 {
  background-color: rgb(255, 255, 0); /* Yellow */
}
```

## HSL
> 참고: HSB(Hue, Saturation, Brightness)라는 유사한 모델도 있지만, CSS 표준은 HSL입니다.

색조(Hue), 채도(Saturation), 명도(Lightness)를 이용해 색상을 표현하는, 보다 인간 친화적인 방식이다.
- 색조 (Hue): 0부터 360까지의 값으로 색상환의 각도를 나타낸다. (0: 빨강, 120: 초록, 240: 파랑)
- 채도 (Saturation): 0%부터 100%까지의 값으로, 0%는 회색톤, 100%는 가장 선명한 순색을 의미한다.
- 명도 (Lightness): 0%부터 100%까지의 값으로, 0%는 검은색, 100%는 흰색을 나타낸다.

<Image src={HSL} alt="HSL 색상" />

```css
.btn-hsl-1 {
  background-color: hsl(248, 83%, 71%);
}

.btn-hsl-2 {
  background-color: hsl(248, 63%, 71%);
}

.btn-hsl-3 {
  background-color: hsl(248, 43%, 71%);
}

.btn-hsl-4 {
  background-color: hsl(248, 23%, 71%);
}
```

## LCH
인간이 색상을 인지하는 방식에 더 가깝게 설계된 색상 모델이다. 
명도(Lightness), 채도(Chroma), 색조(Hue)로 구성된다. HSL과 달리, 명도(Lightness) 값을 변경해도 사람이 느끼는 시각적 밝기가 일정하게 유지되는 '지각적 균일성'을 가진다.
- 명도 (Lightness): 0% ~ 100%
- 채도 (Chroma): 색상의 선명도를 나타낸다. 0부터 시작하며 값이 클수록 색상이 진해진다.
- 색조 (Hue): 0부터 360까지의 값으로 색상환의 각도를 나타낸다. (0: 빨강, 120: 초록, 240: 파랑)

<Image src={LCH} alt="LCH 색상" />

```css
.btn-lch-1 {
  background-color: lch(65% 120 248);
}

.btn-lch-2 {
  background-color: lch(65% 120 200);
}

.btn-lch-3 {
  background-color: lch(65% 120 180);
}

.btn-lch-4 {
  background-color: lch(65% 120 160);
}

.btn-lch-5 {
  background-color: lch(65% 100 248);
}

.btn-lch-6 {
  background-color: lch(65% 80 200);
}

.btn-lch-7 {
  background-color: lch(65% 60 180);
}

.btn-lch-8 {
  background-color: lch(65% 40 160);
}
```

## OKLCH
LCH를 수학적으로 더욱 개선하여, 특히 그라데이션이나 색상 변환 시 발생할 수 있는 색상 왜곡 문제를 해결한 최신 색상 모델이다. 구성 요소는 LCH와 동일하다.

LCH와 OKLCH는 채도와 색조를 고정한 채 명도를 조절하거나, 명도와 색조를 고정한 채 채도를 조절할 때 매우 일관성 있고 예측 가능한 결과를 보여준다. 이는 접근성 높은 UI나 정교한 색상 시스템을 만들 때 매우 유용하다.

<Image src={OKLCH와LCH차이} alt="OKLCH와LCH차이"  />

- 상단이 OKLCH, 하단이 LCH

<Image src={OKLCH} alt="OKLCH 색상"  />

## 고급 사용법 : from 문법 + calc 문법 혼합
`from` 키워드를 사용하면 기준 색상을 지정하고, 그 색상의 l, c, h 값을 변수처럼 가져와 `calc()` 함수로 동적인 색상을 만들 수 있다.

```css{3}
.btn-oklch-9 {
	--base-color: rgb(100, 50, 100); 
	background-color: oklch(from var(--base-color) calc(l + 0.1) c h);
}
```
